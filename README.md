# Kudos Donations

Easily receive donations in WordPress via Mollie

## Development

Use the [yarn](https://yarnpkg.com/) and [composer](https://getcomposer.org/) package managers to install the required dependencies.

```bash
yarn install
composer install
```

## Usage

Development
```bash
yarn start
```

Deployment
```bash
yarn build
```
